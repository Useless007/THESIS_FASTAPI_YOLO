
เว็บแอปพลิเคชัน ที่พัฒนาด้วย FastAPI ซึ่งออกแบบมาเพื่อรองรับการจัดการร้านค้าออนไลน์และกระบวนการจัดการคำสั่งซื้อ โดยมีฟีเจอร์หลักดังนี้:



1. ระบบจัดการร้านค้า
- การจัดการสินค้า: รองรับการเพิ่ม, ลบ, และแก้ไขข้อมูลสินค้า เช่น ชื่อ, ราคา, และรายละเอียด
- ตะกร้าสินค้า: ลูกค้าสามารถเพิ่มสินค้าในตะกร้าและดำเนินการชำระเงินผ่านการแนบสลิปการโอนเงิน
- การตรวจสอบคำสั่งซื้อ: ผู้ดูแลสามารถตรวจสอบคำสั่งซื้อและสลิปการโอนเงินก่อนอนุมัติ



2. ระบบจัดการผู้ใช้
- การลงทะเบียนและล็อกอิน: รองรับการลงทะเบียนผู้ใช้ใหม่และการล็อกอินด้วย JWT Authentication
- บทบาทผู้ใช้: มีการกำหนดบทบาท เช่น ลูกค้า, พนักงาน, ผู้ดูแล และผู้บริหาร เพื่อควบคุมสิทธิ์การเข้าถึงฟีเจอร์ต่าง ๆ
- การรีเซ็ตรหัสผ่าน: ผู้ใช้สามารถรีเซ็ตรหัสผ่านผ่านระบบที่ปลอดภัย



3. ระบบจัดการคำสั่งซื้อ
- พนักงานจัดเตรียมสินค้า: มีแดชบอร์ดสำหรับดูคำสั่งซื้อที่ต้องจัดเตรียม และสามารถยืนยันคำสั่งซื้อได้
- พนักงานบรรจุสินค้า: ใช้เทคโนโลยีการประมวลผลภาพ (Image Processing) เพื่อตรวจสอบความถูกต้องของสินค้าก่อนบรรจุ
- ผู้บริหาร: สามารถดูรายงานสรุปผลการดำเนินงาน เช่น ยอดขายรายวันและประวัติการทำงานของพนักงาน



4. ระบบรักษาความปลอดภัย
- JWT Authentication: ใช้ JSON Web Token ในการยืนยันตัวตนและควบคุมการเข้าถึง API
- Middleware: มีการเพิ่ม Middleware หลายตัว เช่น:
  - AuthRedirectMiddleware: จัดการการยืนยันตัวตน
  - BlockMaliciousRequestsMiddleware: ป้องกันคำขอที่เป็นอันตราย
  - TrustedHostMiddleware: จำกัดการเข้าถึงเฉพาะโดเมนที่เชื่อถือได้



5. การออกแบบและการใช้งาน
- Frontend Integration: ใช้เทมเพลต HTML (เช่น cart.html, home.html) สำหรับการแสดงผลหน้าเว็บ
- CORS Configuration: รองรับการเชื่อมต่อจากโดเมนที่กำหนด เช่น `https://home.jintaphas.tech`
- Static Files: ให้บริการไฟล์ Static เช่น รูปภาพและไฟล์อัปโหลดผ่าน static และ uploads



6. เทคโนโลยีที่ใช้
- FastAPI: สำหรับการพัฒนา Backend
- JWT: สำหรับการยืนยันตัวตน
- HTML + JavaScript: สำหรับการพัฒนา Frontend
- Image Processing: ใช้ในกระบวนการตรวจสอบสินค้าก่อนบรรจุ



สรุป:  
โปรเจกต์นี้เป็นระบบจัดการร้านค้าออนไลน์ที่ครอบคลุมทุกกระบวนการ ตั้งแต่การจัดการสินค้า การสั่งซื้อ การตรวจสอบคำสั่งซื้อ ไปจนถึงการจัดการพนักงานและการรายงานผล เหมาะสำหรับธุรกิจที่ต้องการระบบที่ปลอดภัยและมีประสิทธิภาพสูง
